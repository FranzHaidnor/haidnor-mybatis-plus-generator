# haidnor-mp-generator
集成 mybaits plus 代码生成器

# Maven
```xml
<dependency>
    <groupId>haidnor</groupId>
    <artifactId>haidnor-mybatis-plus-generator</artifactId>
    <version>1.0</version>
    <scope>test</scope>
</dependency>
```

# 实例代码说明
1、MultiModuleGenerator 项目多模块代码生成器项
2、SingleModuleGenerator 项目单模块代码生成器
